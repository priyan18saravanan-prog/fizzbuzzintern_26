# FizzBuzz

A simple Python FizzBuzz package with tests.

## Usage

Run directly with Python:

```bash
python -m fizzbuzz
```

Or, after installing, run the CLI:

```bash
fizzbuzz
```

## Installation

```bash
pip install -e .
```

## Tests

After installing, run:

```bash
python -m unittest discover -s tests
```

Or without installing, use:

```bash
PYTHONPATH=src python -m unittest discover -s tests
```
